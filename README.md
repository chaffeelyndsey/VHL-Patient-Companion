# VHL Companion
## Overview

VHL Companion is a patient-centered tracking application designed for individuals with Von Hippel-Lindau disease. The goal is to help patients organize surveillance imaging, specialist appointments, tumor history, medications, symptoms, and care notes in one centralized system. 

This project is currently in the early design and MVP development phase. 

## Problem Statement

Von Hippel-Lindau disease requires lifelong monitoring across multiple organ systems, including the brain, spine, kidneys, pancreas, adrenal glands, eyes, and inner ear. Patients may need recurring MRI, eye exams, lab work, blood pressure tracking, specialist visits, and tumor growth monitoring. Managing this information across multiple providers can become difficult, especially when surveillance is lifelong and time-sensitive. 

## Project Goals

- Track tumors by organ system, location, size, and imaging device
- Track surveillance appointments and imaging schedules
- Log symptoms over time (Habit tracker style)
- Manage medications and care notes
- Create a patient dashboard for quick health overview
- Build a foundation for future clinical decision-support features
